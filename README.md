# ✦ yassy creates

A cute dark-mode portfolio crafted with immersive animations, interactive 3D visuals, and a soft pastel aesthetic. Built to showcase creative work, frontend development, and modern web experiences.

---

## ✦ Features

- Smooth fullscreen slide transitions
- Interactive 3D background visuals
- Responsive desktop / tablet / mobile layout
- Premium dark aesthetic UI
- Animated portfolio showcase
- Custom contact section
- Soft pastel gradients & glassmorphism styling

---

## ✦ Built With

- HTML5
- CSS3
- Vanilla JavaScript
